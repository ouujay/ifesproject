Current Openings

Junior Web Developer

Location: Hybrid / Remote

Responsibilities:

  • Assist in designing and developing websites
  • Debug and maintain existing code
  • Work closely with senior developers to learn best practices

Qualifications:

  • Basic knowledge of HTML, CSS, and JavaScript
  • Eagerness to learn modern frameworks
  • Good communication and teamwork skills

Mid-level Web Developer

Location: Hybrid / Remote

Responsibilities:

  • Develop and optimize front-end architecture
  • Integrate back-end services and APIs
  • Mentor junior developers

Qualifications:

  • 2-3 years of web development experience
  • Proficient in JavaScript frameworks (React, Vue, or Angular)
  • Experience with version control (Git)

Senior Web Developer

Location: Hybrid / Remote

Responsibilities:

  • Lead complex projects from concept to completion
  • Architect scalable and maintainable systems
  • Guide and mentor junior and mid-level developers

Qualifications:

  • 5+ years of web development experience
  • Expert in one or more modern JavaScript frameworks
  • Strong problem-solving and leadership skills
